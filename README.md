# RiderGo
## Bike and Scooter Rental System

**RiderGo** is a modern web-based platform that allows users to rent bikes and scooters online easily. The system provides a simple, user-friendly interface that lets users browse vehicles, book rides, and manage their bookings efficiently.

This project was developed as part of **Beyond Curriculum Training** to gain hands-on experience in **Full Stack Web Development**.

---

# 🚀 Features

- 🔐 User Authentication (Login / Signup)  
- 🏍️ Browse Available Bikes & Scooters  
- 📅 Easy Ride Booking System  
- 📊 User Dashboard  
- 📜 Ride History / Recent Trips  
- 💳 Wallet & Ride Statistics  
- 🎨 Clean and Responsive UI  

---

# 🛠️ Tech Stack

**Frontend**

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/ridergo.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
